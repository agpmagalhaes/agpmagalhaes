The original idea for the project “Barbearia Alura” came from a practice in the Front End course at Alura. The idea was very interesting, but the initial project was basic, so I decided to improve it in my own way.

This project contains header and footer with background image, banner with image slide, addition of YouTube video and Google Maps map. It also has a contact form and table. Everything was done using HTML5, CSS3 and JavaScript.
