O projeto “Club Pets&Pets” tem o objetivo de explorar os campos de input através de um formulário de cadastro. Os tipos de input usados aqui são checkbox, date, email, file, number, password, radio, range, submit, tel e text, acompanhados de textarea e select. Neste projeto eu mantive o layout o mais simples possível, concentrando o foco no formulário, mas ainda assim trazendo uma aparência mais divertida com a escolha de cores e o background com ilustrações dos pets.

Quanto à responsividade, utilizei bastante o flexbox para que o formulário se adapte à qualquer tamanho de tela, dispensando o uso de @media, as fontes utilizam a unidade de medida “em” para se adaptarem às preferências do usuário. Levando em consideração a usabilidade, separei as áreas de cadastro por cores, para que o usuário não confunda quando deve colocar os dados do pet ou seus próprios dados.
